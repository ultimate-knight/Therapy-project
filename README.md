
# Therapy Task Website

This project is a modern, responsive landing page for Dr. Maya Reynolds, PsyD, a licensed clinical psychologist in Santa Monica. Built with Next.js and React, it showcases her therapy services, specialities, professional background, and contact information.

## Features
- **Responsive Design:** Adapts to all screen sizes, including mobile and desktop.
- **Sectioned Layout:**
	- Hero section with introduction and call-to-action.
	- Overview of therapy approach and benefits.
	- Specialities: Anxiety & Stress, Trauma & Healing, Burnout & Perfectionism.
	- Office details and environment.
	- FAQs with interactive expand/collapse.
	- Professional background and certifications.
	- Contact, hours, and navigation links.
	- Footer with legal and privacy links.
- **Interactive Elements:**
	- Mobile menu toggle.
	- FAQ and background sections with expand/collapse icons.
	- Multiple call-to-action buttons.
- **Image Assets:**
	- Uses local images for visual appeal (office, flowers, portraits, etc).

## Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
	 ```bash
	 git clone <your-repo-url>
	 cd Therapy-task
	 ```
2. Install dependencies:
	 ```bash
	 npm install
	 # or
	 yarn install
	 ```

### Running the Development Server
```bash
npm run dev
# or

```
Visit [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure
- `src/app/page.js`: Main homepage component with all sections.
- `public/`: Contains image assets used throughout the site.
- `app/globals.css`: Global styles.
- `package.json`: Project dependencies and scripts.

## Customization
- Update text, images, and contact details in `src/app/page.js` as needed.
- Replace image files in `public/` for your own branding.

## Dependencies
- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Lucide React](https://lucide.dev/icons) (for icons)
- Tailwind CSS (utility classes)
